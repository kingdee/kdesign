语义化的矢量图形。<br/>

## API

| 属性      | 说明                 | 类型    | 默认值 | 版本  |
| --------- | -------------------- | ------- | ------ | ----- |
| icon-name | 图标名称             |         | -      | 1.0.0 |
| spin      | 是否开启旋转动画效果 | boolean | FALSE  | 1.0.0 |
| onclick   | 点击时触发           |         | -      | 1.0.0 |

## 设计变量

| 类别       | Token名称               | 说明     | 默认值                           |
| ---------- | ----------------------- | -------- | -------------------------------- |
| typography | --kdds-c-icon-font-size | 图标大小 | var(--kdds-g-icon-sizing-3,1rem) |
